Federated Bookkeeping

Definition

Federated Bookkeeping is the technique of electronically linking two more bookkeeping systems, in such a way that accounts, purchase orders, invoices, tax statements, etc. stay in sync without the need to go through the traditional export and send process at one system, followed by the receive and import process at the other system.

Implementations

The MoneyPane which is part of Solid OS is a first implementation of a simplistic bookkeeping system that supports Federated Bookkeeping. It is still under construction. The design consists of several components:

  • Ledger, using Synchronized Network Accounting Protocol (SNAP)
  • Purchase Orders
  • Invoices
